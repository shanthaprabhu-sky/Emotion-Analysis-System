# AI-Powered Emotion Analysis System

## Overview

An AI-powered Emotion Analysis System that detects human emotions from uploaded images and live webcam feeds using DeepFace, OpenCV, TensorFlow, and Streamlit.

## Features

- Emotion Detection from Images
- Real-Time Webcam Emotion Detection
- Confidence Score Display
- Emotion Breakdown Chart
- No Face Detection Handling
- Interactive Web Interface

## Technologies Used

- Python
- Streamlit
- DeepFace
- OpenCV
- TensorFlow

## Installation

```bash
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run app.py
```
